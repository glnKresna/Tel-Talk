Clone repo, kalo udah:
```
cd tel-talk
npm install
```

## Kalo mau ngepush
1. Cek versi kode dulu:
```
git checkout main
git pull origin main
```

2. Buat branch:
```
git checkout -b (nama)/(ngapain)
```

3. Kalo udah selesai nambahin/ngedit kode, commit:
```
git add .
git commit -m "Terserah apa yg penting jelasin"
```

4. Push kode yg baru:
```
git push -u origin (nama)/(ngapain)
```
